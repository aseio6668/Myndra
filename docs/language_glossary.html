Core Language Elements

Keyword
fn
fn function_name(parameters) -> return_type { }
Defines a function with optional parameters and return type.
Example:
fn add(a: int, b: int) -> int { return a + b }
Keyword
let
let variable_name = value
Declares a variable with type inference or explicit typing.
Example:
let name = "Alice"
let age: int = 25
Keyword
import
import module_name with capabilities(capability_list)
Imports a module with specific capability requirements for security.
Example:
import std::io with capabilities("read", "write")
Keyword
match
match value { pattern => result }
Pattern matching for control flow and value extraction.
Example:
match age { 0..18 => "minor", 18..65 => "adult", 65.. => "senior" }

Data Types

Type
int
let number: int = 42
Integer type for whole numbers.
Example:
let count: int = 100
Type
float
let number: float = 3.14
Floating-point number type for decimal values.
Example:
let pi: float = 3.14159
Type
string
let text: string = "Hello"
Text string type for character data.
Example:
let message: string = "Welcome to Pomegranate!"
Type
bool
let flag: bool = true
Boolean type for true/false values.
Example:
let is_active: bool = false
Type
[Type]
let array: [int] = [1, 2, 3]
Array type for collections of elements of the same type.
Example:
let names: [string] = ["Alice", "Bob", "Charlie"]

Control Flow

Control Flow
if/else
if condition { } else { }
Conditional execution based on boolean expressions.
Example:
if age >= 18 { print("Adult") } else { print("Minor") }
Control Flow
for
for variable in iterable { }
Loop over collections or ranges.
Example:
for i in 0..10 { print(i) }
for name in names { print(name) }
Control Flow
while
while condition { }
Loop while condition is true.
Example:
while counter < 10 { counter += 1 }
Control Flow
loop
loop { }
Infinite loop, use break to exit.
Example:
loop { if should_exit { break } }

Functions & Methods

Function
return
return value
Returns a value from a function.
Example:
fn double(x: int) -> int { return x * 2 }
Function
fallback
fn name() fallback strategy { }
Defines fallback behavior for error handling.
Example:
fn fetch_data() fallback retry(3) or return default_data() { }
Function
main
fn main() { }
Program entry point function.
Example:
fn main() { print("Hello World!") }

Reactive Programming

Reactive System Components

Reactive
@reactive
@reactive fn function_name() { }
Annotation that makes a function reactive to data changes.
Example:
@reactive fn update_ui() { /* reacts to data changes */ }
Reactive
observable
let obs = observable<Type>(initial_value)
Creates a reactive data stream that can be observed.
Example:
let temperature = observable<float>(20.0)
Reactive
subscribe
observable.subscribe { value -> action }
Listens to changes in an observable and executes a callback.
Example:
temp.subscribe { value -> print("Temp: " + value) }
Reactive
emit
observable.emit(new_value)
Sends a new value to all subscribers of an observable.
Example:
temperature.emit(25.0)

Temporal Types

Time-Aware Programming

Temporal
evolving<Type>
let value: evolving<Type> = initial_value
A variable that can change its value over time with smooth transitions.
Example:
let opacity: evolving<float> = 0.0
Temporal
->
variable -> target_value over duration
Animates a temporal variable to a new value over a specified time period.
Example:
opacity -> 1.0 over 2s
position_x -> 100.0 over 500ms
Temporal
then
animation then next_animation
Chains temporal animations to run in sequence.
Example:
scale -> 1.5 over 1s then scale -> 1.0 over 1s
Temporal
@interval
@interval(duration) function_call()
Executes a function repeatedly at specified time intervals.
Example:
@interval(16ms) update_physics()

Live Code Capsules

Capsule
capsule
capsule name { functions and data }
A live code block that can be modified and reloaded at runtime.
Example:
capsule ui_component { fn render() -> Element { } }

Security & Capabilities

Security-First Programming

Security
capabilities
import module with capabilities("cap1", "cap2")
Declares the specific permissions a module requires to operate.
Example:
import net::http with capabilities("fetch", "read-only")
Security
with_capabilities
operation with_capabilities(["cap1", "cap2"])
Executes an operation with specific capability enforcement.
Example:
db::execute(query) with_capabilities(["read"])

Context-Aware Features

Context
context
if context == "environment" { }
Built-in variable that contains the current execution context (dev/prod/test).
Example:
log("Debug info") if context == "dev"
Context
env
env("VARIABLE_NAME")
Accesses environment variables.
Example:
let api_key = env("API_KEY")

Domain-Specific Languages

DSL
dsl
dsl language_name { domain-specific code }
Embeds domain-specific languages directly in Pomegranate code.
Example:
dsl sql { SELECT * FROM users WHERE active = true }
dsl shader { precision mediump float; }

Parallel Computing

Parallel
@parallel
@parallel fn function_name() { }
Annotation that enables automatic parallelization of a function.
Example:
@parallel fn process_data(data: [Item]) -> [Result] { }
Parallel
@async
@async { asynchronous code }
Executes code asynchronously without blocking the main thread.
Example:
@async { let result = fetch_data() }
Parallel
await
let result = await async_operation
Waits for an asynchronous operation to complete and returns its result.
Example:
let data = await fetch_user_data(user_id)

Decentralized Identity

DID
did::verify
did::verify(token)
Verifies a decentralized identity token.
Example:
let user = did::verify(auth_token)
DID
has_proof
identity.has_proof("condition")
Checks if an identity has a zero-knowledge proof for a specific condition.
Example:
if user.has_proof("age > 18") { grant_access() }

Package Management

Package
pkg::registry
pkg::registry::function_name()
Access to decentralized package registry operations.
Example:
pkg::registry::publish(package_hash, manifest)

Error Handling

Error
try/catch
try { code } catch error { handler }
Exception handling for error recovery.
Example:
try { risky_operation() } catch error { log_error(error) }
Error
throw
throw ErrorType("message")
Throws an exception with a specific error type and message.
Example:
throw ValidationError("Invalid input data")
Error
retry
retry(count) or alternative
Retry operation a specified number of times before fallback.
Example:
fallback retry(3) or return default_value()

Annotations

Annotation
#tag
#tag:category
Tags functions or code blocks for organization and navigation.
Example:
#tag:ui
fn render_button() { }
Annotation
@timeout
@timeout(duration) { code }
Sets a timeout for code execution.
Example:
@timeout(5s) { fetch_remote_data() }

Pomegranate3D - 3D Graphics

3D Core
world
world { camera { } lighting { } }
Creates a 3D world container with camera, lighting, and scene objects.
Example:
world { camera { position: vec3(0, 0, 5) } }
3D Type
vec3
vec3(x, y, z)
3D vector for positions, directions, colors. Supports all mathematical operations.
Example:
let position = vec3(1.5, 2.0, -3.0)
let distance = position.magnitude()
3D Type
vec2
vec2(x, y)
2D vector for texture coordinates, screen positions, or 2D math.
Example:
let texture_coord = vec2(0.5, 0.5)
3D Type
quat
quat::from_euler(x, y, z) or quat::identity()
Quaternion for 3D rotations. More efficient and stable than Euler angles.
Example:
let rotation = quat::from_euler(45.degrees, 0, 90.degrees)
3D Type
color
color(r, g, b) or color(r, g, b, a)
RGBA color with values from 0.0 to 1.0. Supports color math operations.
Example:
let red = color(1.0, 0.0, 0.0, 1.0)
let bright_red = red * 1.5
3D Mesh
mesh::cube
mesh::cube { size: vec3, position: vec3, material: Material }
Creates a 3D cube/box mesh with specified dimensions and properties.
Example:
mesh::cube { size: vec3(2, 1, 3), material: color(1, 0.5, 0) }
3D Mesh
mesh::sphere
mesh::sphere { radius: float, subdivisions: int, material: Material }
Creates a 3D sphere mesh with specified radius and smoothness level.
Example:
mesh::sphere { radius: 1.5, subdivisions: 32, material: color(0.2, 0.8, 0.2) }
3D Mesh
mesh::plane
mesh::plane { size: vec2, resolution: (int, int), material: Material }
Creates a flat plane mesh, useful for ground, walls, or water surfaces.
Example:
mesh::plane { size: vec2(10, 10), material: color(0.7, 0.7, 0.7) }
3D Camera
camera
camera { position: vec3, target: vec3, fov: degrees }
Defines the viewpoint for the 3D scene with position, target, and field of view.
Example:
camera { position: vec3(0, 5, 10), target: vec3(0, 0, 0), fov: 60.degrees }
3D Lighting
light::directional
light::directional { direction: vec3, color: color, intensity: float }
Directional light source (like the sun) that illuminates from a specific direction.
Example:
light::directional { direction: vec3(-1, -1, -1), color: color(1, 0.9, 0.8), intensity: 2.0 }
3D Lighting
light::point
light::point { position: vec3, color: color, intensity: float, range: float }
Point light source (like a lightbulb) that emits light in all directions.
Example:
light::point { position: vec3(3, 4, 2), color: color(1, 0.8, 0.6), intensity: 5.0, range: 10.0 }
3D Material
materials::pbr
materials::pbr { albedo: color, metallic: float, roughness: float }
Physically-Based Rendering material for realistic surface appearance.
Example:
materials::pbr { albedo: color(0.7, 0.1, 0.1), metallic: 0.8, roughness: 0.2 }
3D Material
materials::glass
materials::glass { color: color, transparency: float, refraction_index: float }
Glass material with transparency and light refraction effects.
Example:
materials::glass { color: color(0.9, 0.9, 1.0, 0.3), refraction_index: 1.5 }
3D Animation
animate
animate(object).rotate(axis, degrees).duration(time).loop()
Declarative animation system for smooth object motion and transformation.
Example:
animate(cube).rotate(axis: y, degrees: 360).duration(3s).loop()
3D Animation
timeline
timeline name { at(time): { properties }, ease: easing_function }
Keyframe-based animation system with automatic interpolation between states.
Example:
timeline dance { at(0s): obj.position = vec3(0,0,0), at(2s): obj.position = vec3(5,0,0) }
3D Animation
behavior
behavior "name" { parameters: {}, motion: {} }
Reusable animation behavior that can be applied to any 3D object.
Example:
behavior "hover" { motion: { translate(by: vec3(0, sin(time), 0)) } }
3D Shader
shader
shader name { vertex { }, fragment { } }
Custom shader definition with vertex and fragment stages integrated in the language.
Example:
shader waves { vertex { vertex.position.y += sin(vertex.position.x + time) } }
3D Physics
physics
physics: { type: dynamic, mass: float, collision_shape: shape }
Physics properties for 3D objects enabling realistic collision and movement.
Example:
physics: { type: dynamic, mass: 1.0, collision_shape: box }
3D Particles
particle_system
particle_system name { emission_rate: rate, lifetime: duration, material: Material }
Advanced particle system for fire, smoke, magic effects, and environmental particles.
Example:
particle_system fire { emission_rate: 100.per_second, lifetime: 2s, material: materials::flame }
3D Procedural
procedural_terrain
procedural_terrain { size: vec2, height_generation: {}, biomes: [] }
Generates infinite, varied terrain using noise functions and biome systems.
Example:
procedural_terrain { size: vec2(1000, 1000), height_generation: { base_noise: fbm_noise(scale: 0.01) } }
3D Procedural
l_system
l_system name { axiom: "F", rules: { "F" => "F[+F]F[-F]F" } }
Lindenmayer system for procedural generation of trees, plants, and organic structures.
Example:
l_system tree { axiom: "F", rules: { "F" => "F[+F][-F]" }, iterations: 4 }
3D Noise
fbm_noise
fbm_noise(position, octaves: int, persistence: float)
Fractal Brownian Motion noise for natural-looking procedural generation.
Example:
let height = fbm_noise(vec2(x, z) * 0.01, octaves: 6, persistence: 0.5)
3D Render
render_world
render_world()
Starts the 3D rendering loop with automatic optimization and cross-platform support.
Example:
render_world() // Starts rendering at optimal framerate

Built-in Functions

Built-in
print
print(value)
Outputs a value to the console.
Example:
print("Hello World!")
Built-in
log
log(message)
Logs a message with appropriate formatting and context.
Example:
log("User logged in: " + user.name)
Built-in
now
now()
Returns the current timestamp.
Example:
let timestamp = now()
Built-in
sleep
sleep(duration)
Pauses execution for a specified duration.
Example:
sleep(100ms)
sleep(2s)
Built-in
.to_string()
value.to_string()
Converts any value to its string representation.
Example:
let text = age.to_string()
Built-in
.map()
array.map { item -> transformation }
Transforms each element in a collection.
Example:
let doubled = numbers.map { n -> n * 2 }
Built-in
.filter()
array.filter { item -> condition }
Filters elements based on a condition.
Example:
let adults = people.filter { person -> person.age >= 18 }
Built-in
.reduce()
array.reduce(initial_value, { acc, item -> operation })
Reduces a collection to a single value using an accumulator function.
Example:
let sum = numbers.reduce(0, { acc, n -> acc + n })
