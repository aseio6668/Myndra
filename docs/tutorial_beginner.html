1. Your First Myndra Program

Let's start with the classic "Hello World" program. In Myndra, you can create context-aware programs right from the beginning:

// hello_world.pom
import std::io with capabilities("write")

fn main() {
    log("Starting Myndra program") if context == "dev"
    
    let message = "Hello, Myndra!"
    print(message)
    
    log("Program completed successfully") if context == "dev"
}
üéØ Context-Aware Feature: Notice the if context == "dev" syntax. This means the log statements only execute in development mode, not in production!

2. Basic Syntax Elements

Variables and Data Types

// Basic variable declaration
let name = "Alice"              // String
let age = 25                    // Integer  
let height = 5.6                // Float
let is_student = true           // Boolean

// Arrays
let numbers = [1, 2, 3, 4, 5]
let names = ["Alice", "Bob", "Charlie"]

Functions

// Simple function
fn greet(name: string) -> string {
    return "Hello, " + name + "!"
}

// Function with multiple parameters
fn calculate_area(length: float, width: float) -> float {
    return length * width
}

// Using functions
fn main() {
    let greeting = greet("World")
    print(greeting)
    
    let area = calculate_area(10.0, 5.0)
    print("Area: " + area.to_string())
}

3. Unique Myndra Features

Capability-Based Imports

Myndra modules declare exactly what permissions they need:

// Only import what you need with specific permissions
import net::http with capabilities("fetch", "read-only")
import std::file with capabilities("read")
import std::ui with capabilities("render", "event-handling")
üîí Security Feature: This prevents modules from accessing resources they shouldn't have, making your programs more secure by default.

Tags for Code Organization

// Use tags to organize and navigate your code
#tag:math
fn add(a: int, b: int) -> int {
    return a + b
}

#tag:ui  
fn create_button(text: string) -> Element {
    return ui::button(text)
}

#tag:data

    // Load user data
}

4. Working with User Input

import std::io with capabilities("read", "write")

fn main() {
    print("What's your name? ")
    let name = io::read_line()
    
    print("How old are you? ")
    let age_str = io::read_line()
    let age = age_str.to_int()
    
    if age >= 18 {
        print("Hello " + name + ", you're an adult!")
    } else {
        print("Hello " + name + ", you're still young!")
    }
}

5. Control Flow

If Statements

let score = 85

if score >= 90 {
    print("Excellent!")
} else if score >= 80 {
    print("Good job!")
} else if score >= 70 {
    print("Not bad!")
} else {
    print("Keep trying!")
}

Loops

// For loop with range
for i in 0..5 {
    print("Count: " + i.to_string())
}

// For loop with array
let fruits = ["apple", "banana", "orange"]
for fruit in fruits {
    print("I like " + fruit)
}

// While loop
let counter = 0
while counter < 5 {
    print("Counter: " + counter.to_string())
    counter = counter + 1
}

6. Error Handling with Fallbacks

Myndra has built-in self-healing error handling:

fn divide(a: float, b: float) -> float fallback return 0.0 {
    if b == 0.0 {
        throw DivisionByZeroError("Cannot divide by zero")
    }
    return a / b
}

fn fetch_data(url: string) -> Data 
    fallback retry(3) or return default_data() {
    // This will retry 3 times, then return default data if it still fails
    let response = http::get(url)
    return parse_data(response)
}
